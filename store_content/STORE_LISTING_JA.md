## 詳しい説明（4,000文字以内）

```
純粋なロジックの楽しさ、Ninedoku。

広告なし。インターネット不要。数独だけに集中。
Ninedokuは、美しいデザインの完全オフライン数独アプリです。

◆ あなたのレベルに合わせて
初心者からエキスパートまで、6段階の難易度で最適なチャレンジを。

• 入門 — 数独が初めての方
• 初級 — 基本ルールを知っている方
• 中級 — 基本テクニックができる方
• 上級 — 高度な戦略を活用する方
• 難問 — 難しいパズルを楽しむ方
• 達人 — 究極の挑戦を求める方

◆ 多彩なゲームモード
クラシック数独に加え、対角線数独やブロック数独など、バリエーション豊富なモードをお楽しみください。

◆ 毎日の新パズル
デイリーパズルで毎日新しい挑戦。継続プレイで実力アップ、連続記録も積み重ねましょう。

◆ 成長を実感
詳細な統計でゲーム時間、最高記録、クリア率を追跡。グラフで見る成長過程がモチベーションに。

◆ バッジコレクション
様々なチャレンジを達成してバッジを集めましょう。初クリアから連続達成まで、特別なバッジが待っています。

◆ 完全オフライン
インターネット接続不要。飛行機の中でも、地下鉄でも、どこでもプレイ可能。インターネット権限自体がないため、データの心配も不要です。

◆ 広告なし・トラッキングなし
広告は一切ありません。分析SDK、トラッキングツールなど、データ収集ツールは使用していません。

◆ 8言語対応
한국어, English, 日本語, 中文, हिन्दी, Español, العربية, Français

◆ 主な機能
✓ 6段階の難易度（入門〜達人）
✓ 多彩なゲームモード
✓ デイリーパズル
✓ 詳細統計＆チャート
✓ バッジ・実績システム
✓ メモ機能
✓ 元に戻す/やり直し
✓ 完全オフライン
✓ 広告なし
✓ 個人情報収集なし
✓ 8言語対応
```
