## IARC 설문 답변 가이드

Play Console → 앱 콘텐츠 → 콘텐츠 등급 → 설문 시작

### 카테고리 선택

**Q: 앱의 카테고리는 무엇인가요?**
→ **게임** (퍼즐)

### 등급 근거

Ninedoku는 순수한 논리 퍼즐 게임으로:
- 폭력, 성적 콘텐츠, 도박 요소가 전혀 없음
- 사용자 간 상호작용 기능 없음 (완전 싱글 플레이)
- 인앱 구매 없음
- 광고 없음
- 개인정보 수집 없음
- 모든 연령층이 안전하게 이용 가능

---

### Play Console 입력 절차

1. Play Console → 앱 → **앱 콘텐츠** → **콘텐츠 등급**
2. "설문 시작" 클릭
3. 카테고리: **게임** 선택
4. 위 답변대로 모든 항목 **아니요** 선택
5. 저장 → 등급 확인 → 적용
