## 촬영 절차

```
1. APK 빌드: flutter build apk --release
2. 기기 설치: adb install -r app-release.apk
3. 앱 실행 → 한국어 설정
4. 각 Shot별 화면 이동 후 캡처
5. ADB pull로 PC로 전송
6. store_content/screenshots/ 폴더에 저장
```

---

## 촬영 시 주의사항

- 상태 바 시간을 깔끔한 시간 (10:00 등)으로 맞추기 (DemoMode 활용)
- 배터리 아이콘이 충분한 상태로 보이도록
- 알림 바가 깔끔한 상태 (알림 지우기)
- 게임 보드가 자연스럽게 진행된 상태 (인위적이지 않도록)
