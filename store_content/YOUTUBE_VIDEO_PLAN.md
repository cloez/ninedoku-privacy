## 스토리보드

### Scene 1: 오프닝 (0~4초)
- **화면**: 앱 아이콘 + "Ninedoku" 로고 페이드인
- **텍스트**: "순수한 논리의 즐거움"
- **전환**: 부드러운 페이드

### Scene 2: 홈 화면 (4~8초)
- **화면**: 앱 메인 화면 스크린샷
- **텍스트**: "깨끗한 디자인"
- **전환**: 슬라이드 좌측

### Scene 3: 난이도 선택 (8~12초)
- **화면**: 6단계 난이도 선택 화면
- **텍스트**: "6단계 난이도"
- **전환**: 슬라이드 좌측

### Scene 4: 게임 플레이 (12~18초)
- **화면**: 실제 게임 플레이 화면 (가장 긴 구간)
- **동작**: 숫자 입력 → 검증 → 진행 애니메이션
- **텍스트**: 없음 (게임 화면 자체로 충분)
- **전환**: 크로스 디졸브

### Scene 5: 게임 완료 (18~22초)
- **화면**: 클리어 축하 화면
- **텍스트**: "달성의 즐거움"
- **전환**: 줌 인

### Scene 6: 기능 소개 (22~26초)
- **화면**: 일일 퍼즐 → 통계 → 배지 (빠르게 전환)
- **텍스트**: "매일 새로운 퍼즐 · 통계 · 배지"
- **전환**: 빠른 슬라이드

### Scene 7: 엔딩 CTA (26~30초)
- **화면**: 앱 아이콘 + 핵심 키워드
- **텍스트**:
  - "광고 없음 · 완전 오프라인"
  - "지금 무료 다운로드"
- **배지**: Google Play 배지
- **전환**: 페이드 아웃

## 제작 방식

HTML + CSS 애니메이션으로 제작 후 화면 녹화:

1. 스크린샷 8장을 배경으로 사용
2. CSS @keyframes로 전환 애니메이션 구현
3. 텍스트 오버레이 순차 표시
4. 30초 타임라인에 맞춰 자동 재생
5. Chrome 화면 녹화로 MP4 변환
