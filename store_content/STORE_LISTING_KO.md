## 짧은 설명 (80자 이내)

```
광고 없는 오프라인 스도쿠. 6단계 난이도, 매일 새로운 퍼즐, 통계와 배지까지.
```

## 자세한 설명 (4,000자 이내)

```
순수한 논리의 즐거움, Ninedoku.

광고 없이, 인터넷 없이, 오직 스도쿠에만 집중하세요.
Ninedoku는 깨끗하고 세련된 디자인의 완전 오프라인 스도쿠 앱입니다.

◆ 당신의 수준에 맞는 난이도
초보자부터 전문가까지, 6단계 난이도를 제공합니다.
처음 시작하는 분도 튜토리얼로 쉽게 배울 수 있고,
숙련된 플레이어는 전문가 난이도에서 도전을 즐길 수 있습니다.

• 입문 — 스도쿠를 처음 접하는 분
• 초급 — 기본 규칙에 익숙한 분
• 중급 — 기본 테크닉을 구사하는 분
• 상급 — 고급 전략을 활용하는 분
• 고급 — 어려운 퍼즐을 즐기는 분
• 전문가 — 최고의 도전을 원하는 분

◆ 다양한 게임 모드
클래식 스도쿠 외에도 대각선 스도쿠, 블록 스도쿠 등 다양한 변형 모드를 즐겨보세요. 같은 스도쿠라도 새로운 규칙으로 색다른 재미를 느낄 수 있습니다.

◆ 매일 새로운 퍼즐
일일 퍼즐로 매일 새로운 도전을 만나보세요. 꾸준히 풀면서 실력을 키우고, 연속 플레이 기록도 쌓아가세요.

◆ 나의 실력을 한눈에
상세한 통계 기능으로 게임 시간, 최고 기록, 클리어율을 추적하세요. 그래프로 보는 나의 성장 과정이 동기부여가 됩니다.

◆ 배지 수집의 재미
다양한 도전 과제를 달성하고 배지를 모아보세요. 첫 클리어부터 연속 달성까지, 특별한 배지가 당신을 기다립니다.

◆ 완전 오프라인
인터넷 연결이 필요 없습니다. 비행기 안에서, 지하철에서, 어디서든 즐기세요. 인터넷 권한 자체가 없는 앱이라 데이터 걱정도 없습니다.

◆ 광고 없음 · 추적 없음
광고가 전혀 없습니다. 분석 SDK, 추적 도구 등 어떤 데이터 수집 도구도 사용하지 않습니다. 순수하게 게임만 즐기세요.

◆ 8개 언어 지원
한국어, English, 日本語, 中文, हिन्दी, Español, العربية, Français

◆ 주요 기능 요약
✓ 6단계 난이도 (입문~전문가)
✓ 다양한 게임 모드
✓ 일일 퍼즐
✓ 상세 통계 및 차트
✓ 배지·업적 시스템
✓ 메모 기능
✓ 실행 취소/다시 실행
✓ 완전 오프라인
✓ 광고 없음
✓ 개인정보 수집 없음
✓ 8개 언어 지원
```
