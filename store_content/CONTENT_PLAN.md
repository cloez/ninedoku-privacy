## 3. 스크린샷 촬영 계획

### 촬영 순서 (8장)
1. **홈 화면** — 앱 메인 화면, 브랜드 첫인상
2. **게임 플레이** — 스도쿠 중간 진행 모습, 핵심 콘텐츠
3. **난이도 선택** — 6단계 난이도, 접근성 어필
4. **게임 모드** — 다양한 모드(클래식, 대각선 등)
5. **게임 완료** — 클리어 축하 화면, 성취감
6. **일일 퍼즐** — 매일 새로운 도전, 리텐션 포인트
7. **통계** — 실력 추적, 차트/그래프
8. **배지** — 업적 시스템, 수집 동기

### 촬영 환경
- 실기기 설치 (USB 연결, ADB install)
- 해상도: 기기 원본 (1080×2400 이상 권장)
- 캡처 방식: ADB screencap 또는 기기 스크린샷

## 4. 유튜브 영상 제작 계획

### 영상 사양
| 항목 | 값 |
|------|-----|
| 길이 | 30초~1분 |
| 해상도 | 1920×1080 (16:9) 또는 1080×1920 (9:16) |
| 형식 | MP4 (H.264) |
| 음악 | 로열티 프리 BGM |

### 구성 (30초 버전)
1. **[0-5초]** 앱 아이콘 + "Ninedoku" 타이틀
2. **[5-10초]** 홈 화면 → 게임 시작
3. **[10-18초]** 게임 플레이 하이라이트 (숫자 입력, 검증)
4. **[18-22초]** 게임 완료 축하 애니메이션
5. **[22-26초]** 일일 퍼즐 + 통계 + 배지
6. **[26-30초]** CTA — "지금 무료 다운로드" + Play Store 배지
