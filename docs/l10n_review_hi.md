# हिन्दी (_hi) लोकलाइज़ेशन समीक्षा

## 1. समग्र स्कोर
**7.2 / 10** — मूल कोर अनुवाद स्वाभाविक और सुसंगत, परन्तु बैज/ट्यूटोरियल/Binairo भाग बड़े पैमाने पर अंग्रेज़ी fallback में अटके हैं।

## 2. उत्कृष्ट श्रेणियाँ
- **Home / Difficulty / Mode / Pause / Result** — संक्षिप्त, मित्रवत, मोबाइल-अनुकूल।
- **Settings / Stats** — देवनागरी एकरूपता अच्छी, लम्बाई संतुलित।
- **Daily Puzzle / Weekday संक्षेप** — स्थानीय कैलेंडर परंपरा के अनुरूप।
- खेल शब्दावली: *सुडोकू*, *बैज*, *थीम*, *बैकअप* जैसे लोनवर्ड्स उचित।

## 3. P0 — तुरंत सुधार (≤5)

| Key | वर्तमान | सुझाव |
|---|---|---|
| `badge.firstClear` ~ `badge.streak30` (14 कुंजियाँ) | अंग्रेज़ी ("First Step", "Self Solver" …) | हिन्दी अनुवाद आवश्यक — जैसे *पहला कदम*, *स्वयं सॉल्वर*, *परफेक्शनिस्ट*, *स्पीड रनर*, *उत्साही*, *दिग्गज*, *मास्टर* |
| `tutorial.page1`~`page6` (शीर्षक+विवरण) | पूरी तरह अंग्रेज़ी | हिन्दी ट्यूटोरियल आवश्यक — नए खिलाड़ी के लिए सबसे क्रिटिकल स्क्रीन |
| `binairo.*` (≈30 कुंजियाँ) | पूरी तरह अंग्रेज़ी | कम-से-कम UI बटन (Back/Pause/Resume/New Game/Hub) हिन्दी में: *वापस / रोकें / जारी रखें / नया खेल / हब* |
| `sudoku.about.*`, `sudoku.rules.*` | अंग्रेज़ी | *सुडोकू क्या है?* / *नियम* — हिन्दी अनुवाद |
| `home.progress` = `'% पूर्ण'` | placeholder क्रम अप्राकृतिक (`50% पूर्ण` ठीक, परन्तु key prefix-less) | format string `'{n}% पूर्ण'` स्पष्ट करें; वर्तमान में leading space नहीं |

## 4. P1 — क्रमिक सुधार (≤10)

1. `appTitle` = "K-पज़ल" → ब्रांड शायद ठीक, परन्तु *के-पज़ल* देवनागरी में अधिक सहज।
2. `home.subtitle` "आराम से, एक कक्ष एक बार में" → *आराम से, एक-एक कक्ष* अधिक स्वाभाविक।
3. `game.cellFirst` = "कक्ष", `game.numberFirst` = "अंक" → टॉगल लेबल अस्पष्ट; *कक्ष पहले / अंक पहले*।
4. `game.autoMemo` = "स्वतः" → अकेला शब्द unclear; *स्वतः नोट्स*।
5. `pause.home` = "होम (स्वतः सहेजें)" → *होम (स्वतः सेव)* या *होम — सहेजकर*।
6. `result.grade.s/a/b` स्ट्रिंग्स बहुत खंडित — concat के बाद हिन्दी क्रम टूटता है; पूर्ण वाक्य templating बेहतर।
7. `theme.cafe.desc` "गर्म भूरा आरामदायक थीम" → *गर्म भूरी आरामदायक थीम* (लिंग सहमति)।
8. `theme.dark.desc` "आँखों के लिए आसान" → *आँखों को राहत देने वाली डार्क थीम*।
9. `stats.streak` = "लगातार" → अकेला adverb; *लगातार दिन* या *स्ट्रीक*।
10. `daily.challenge` "आज की पहेली का चुनौती" → लिंग त्रुटि; *आज की पहेली की चुनौती*।

## 5. टोन एकरूपता
- कोर भाग में **औपचारिक "आप" + क्रिया का आदेश रूप** (करें / चुनें / रखें) सुसंगत।
- मिश्रण-समस्या: कोर हिन्दी + ट्यूटोरियल/बैज/Binairo अंग्रेज़ी = उपयोगकर्ता को "अधूरा अनुवाद" का एहसास।
- लोनवर्ड नीति स्पष्ट करें: *बैज, थीम, बैकअप, परफेक्ट* ✅; परन्तु *Hub, Pause, Hints* को भी देवनागरी ध्वनि-अनुवाद करें।
