- **新游戏 12 款标题**：`扫雷 / 数织 / 杀手数独 / 摩天大楼 / 星空对决 / 阴阳` 全部命中标准中文译名。

## 5. 语气一致性
- 主语气：**简洁中性**，符合休闲解谜品类。
- 不一致点：
  - **错误 ↔ 失误** 两词混用（旧版 `数独=错误`，新版 12 款=`失误`）→ 全局统一为"错误"。
  - 标题问号：`giveUp.title=确定放弃？` 与新键 `放弃吗？` 风格不一 → 推荐统一 `确定放弃？`。
  - 退出文案 `离开 / 退出 / 返回` 三词并存，建议：弹窗动作=`离开`，应用退出=`退出`，导航=`返回`（当前大致如此，仅需小幅校正）。
- 字数：所有按钮 ≤4 字，移动端无截断风险。
