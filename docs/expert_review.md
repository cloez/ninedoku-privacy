## R0 Sudoku

### E1: 논리퍼즐 마스터
- ❌ 부족: 체크포인트(되돌릴 분기점) 없음, 같은 숫자 자동 하이라이트 토글 미제공
- ✅ 좋음: 자동 메모(autoNote), 행/열/박스 하이라이트, 메모 모드, 오류 강조 모두 구현
- 💡 개선안: 키 카운터(이미 9개 채운 숫자는 패드 비활성/회색 처리) 강화 + 체크포인트 1개 슬롯 추가

### E2: 모바일 UX 전문가
- ❌ 부족: 마지막 입력 셀 강조(flash) 부재, 완성된 숫자 패드 자동 fade 약함
- ✅ 좋음: number_pad_widget이 숫자별 잔여 카운트 표시함
- 💡 개선안: 입력 직후 0.4s pulse 애니메이션 + 9개 완료 시 패드 버튼 50% opacity

### E3: 접근성 전문가
- ❌ 부족: 색맹 대응 패턴(빨강 오류 외) 부재, 폰트 스케일 옵션 단순
- ✅ 좋음: 시스템 다크모드 연동
- 💡 개선안: 오류 셀에 사선 패턴 병행 표시, 글자 크기 3단계 슬라이더

### 종합 우선순위
- P0: 마지막 변경 강조(공통 적용 기반)
- P1: 체크포인트 1슬롯
- P1: 색맹 패턴 오버레이
- P2: 숫자 패드 완료 fade 강화
- P2: 폰트 크기 슬라이더

## R1 Binairo

### E1: 논리퍼즐 마스터
- ❌ 부족: 완성된 행/열 자동 회색 처리 없음, 행별 0/1 카운터 미표시
- ✅ 좋음: 입력 모드(흑/백/지우개) 토글, 힌트 시스템 견고
- 💡 개선안: 각 행/열 끝에 "검정 N/M · 흰 N/M" 카운터 칩, 동수 도달 시 해당 행 dim

### E2: 모바일 UX 전문가
- ❌ 부족: 행/열 헤더 카운터 없음, 현재 행/열 호버 강조 약함
- ✅ 좋음: 큰 셀 터치 영역, 모드 전환 토글 UX 명료
- 💡 개선안: 셀 선택 시 행/열 전체 옅은 배경 적용

### E3: 접근성 전문가
- ❌ 부족: 흑/백 원이 색맹 사용자에게 명도 차만 의존
- ✅ 좋음: 큰 글자/심볼
- 💡 개선안: 흑=●, 백=○ 외에 텍스처/테두리 차이 보강

### 종합 우선순위
- P0: 행/열 완료 카운터 + 자동 회색
- P1: 행/열 선택 하이라이트
- P1: 마지막 변경 강조
- P2: 색맹 텍스처
- P2: 체크포인트

## R2 Minesweeper

### E1: 논리퍼즐 마스터
- ❌ 부족: 숫자 셀 좌클릭 "체인 오픈"(이미 깃발이 정확히 놓인 경우 주변 자동 오픈) 미구현
- ✅ 좋음: 깃발 모드, 의심(?) 마크, 잔여 지뢰 카운터
- 💡 개선안: chord-click(숫자=깃발수 시 인접 오픈), 안전 첫 클릭 보장 명시

### E2: 모바일 UX 전문가
- ❌ 부족: 롱프레스 깃발 토글 vs 모드 토글 양립 불명, 줌 부재
- ✅ 좋음: 깃발/오픈 모드 분리, 잔여 카운터
- 💡 개선안: 핀치 줌 + 큰 보드(상급)용 미니맵, 진동 햅틱 강화

### E3: 접근성 전문가
- ❌ 부족: 숫자 1~8 색상 외 폰트 두께 차 없음
- ✅ 좋음: 폭발 셀 즉시 표시
- 💡 개선안: 1=가는체, 8=굵은체로 명도 차 보강

### 종합 우선순위
- P0: 체인 오픈(chord)
- P1: 핀치 줌
- P1: 마지막 변경 강조
- P2: 1~8 폰트 두께 차등
- P2: 체크포인트(스냅샷)

## R3 Yin-Yang

### E1: 논리퍼즐 마스터
- ❌ 부족: 영역 분리 위반 즉시 강조 부족, 같은 색 클러스터 표시 없음
- ✅ 좋음: 흑백 토글 + 빈칸, 힌트 솔버
- 💡 개선안: 2x2 동색 블록(룰 위반) 자동 빨강 강조, 색별 연결 그룹 외곽선

### E2: 모바일 UX 전문가
- ❌ 부족: 드래그 페인팅(연속 입력) 없음
- ✅ 좋음: 명확한 흑/백/공백 사이클
- 💡 개선안: 드래그 시 같은 색 연속 칠하기 모드

### E3: 접근성 전문가
- ❌ 부족: 흑/백만으로 구분 → 색맹은 OK지만 저시력자에 작음
- ✅ 좋음: 단순 시각
- 💡 개선안: 셀 내부 ● / ○ 심볼 옵션

### 종합 우선순위
- P0: 2x2 룰 위반 즉시 강조
- P1: 드래그 페인팅
- P1: 연결 그룹 외곽선
- P2: 심볼 모드
- P2: 마지막 변경 강조

## R4 Nonograms

### E1: 논리퍼즐 마스터
- ❌ 부족: 완성된 줄 자동 X(자동 크로스) 미구현, 줄 클리어 시 헤더 dim 없음
- ✅ 좋음: 좌표 표시, 행/열 힌트 막대
- 💡 개선안: 줄 합계 도달 시 빈 칸 자동 X 토글, 완성된 헤더 회색

### E2: 모바일 UX 전문가
- ❌ 부족: 드래그 다중 셀 칠하기 부재, 줌 부재
- ✅ 좋음: 채움/X/빈 사이클
- 💡 개선안: 드래그 페인트 + 핀치 줌 (대형 그리드 필수)

### E3: 접근성 전문가
- ❌ 부족: 채움 색상이 다크모드에서 대비 약함
- ✅ 좋음: 큰 헤더 글자
- 💡 개선안: 채움 셀 고대비 옵션, 5열/10열 굵은 경계선 토글

### 종합 우선순위
- P0: 자동 X (완성된 줄)
- P0: 드래그 페인팅
- P1: 핀치 줌
- P1: 5칸 굵은 경계
- P2: 체크포인트

- ❌ 부족: 라이브 합계(현재 입력 합/목표) 표시 약함, 가능한 조합 힌트 없음
- ✅ 좋음: 메모, 충돌 검사
- 💡 개선안: 각 합계 단서 옆에 "현재합/목표" 라이브 표시, 조합 도우미(예: 합 7/2칸 = {1,6}{2,5}{3,4})

### E2: 모바일 UX 전문가
- ❌ 부족: 합계 단서 글자 작음, 단서 셀 분할 대각선 가독성 낮음
- ✅ 좋음: 메모 모드
- 💡 개선안: 단서 폰트 크기 ↑, 단서 탭 시 해당 행/열 라인 강조

### E3: 접근성 전문가
- ❌ 부족: 검정 단서 셀 + 흰 글자 대비 약함
- ✅ 좋음: 셀 구분 명확
- 💡 개선안: 단서 셀 고대비 모드, 폰트 크기 옵션

### 종합 우선순위
- P0: 라이브 합/잔여 표시
- P0: 조합 도우미
- P1: 단서 폰트 크기 ↑
- P1: 마지막 변경 강조
- P2: 체크포인트

## 13게임 공통 개선 항목 TOP 5 (임팩트 순)

### 1. [P0] 마지막 변경 셀 강조 (Last-Change Highlight)
- 모든 게임에 0.4초 펄스/테두리 애니메이션 공통 적용
- 실수 추적 + 시각 피드백 강화 (참조 사이트 전 게임 기본 제공)
- 영향 게임: 13/13
- 공통 위젯 1개 추가로 일괄 적용 가능

### 2. [P0] 자동 완성 표시 (Auto-Complete Visual Dim)
- 완성된 행/열/숫자/영역을 자동 회색/dim 처리
- 게임별 적용: 노노그램 자동 X / 빈티 자동 잔디 / Light Up 자동 X / Star Battle 자동 X / Binairo 행 dim / Sudoku 패드 dim 등
- 인지 부하 대폭 감소, 풀이 속도 향상

### 3. [P0] 체크포인트 (Checkpoint Save/Restore)
- 모든 13게임에 1슬롯 체크포인트 추가 (게임 중 1회 저장 + 복귀)
- Try-and-error 전략 지원 → 고난도 퍼즐 접근성 ↑
- 참조 사이트 전 게임 공통 기능

### 4. [P1] 핀치 줌 + 큰 그리드 패닝
- 영향 게임: Minesweeper(중상급), Nonograms(15x15+), Star Battle(10x10), Skyscrapers(7x7+)
- 작은 화면에서 큰 보드 가독성 해결

### 5. [P1] 색맹/접근성 패턴 모드
- 오류=빨강 단독 의존 → 사선/패턴 병행
- 영역 구분 색 단독 → 패턴 옵션 (Star Battle, Jigsaw, Killer)
- 폰트 크기 슬라이더(소/중/대) 글로벌 설정

## 권장 실행 순서

1. **공통 위젯 R-Wave**: 마지막 변경 강조 + 체크포인트 시스템 (공통 컴포넌트로 13게임 일괄 적용)
