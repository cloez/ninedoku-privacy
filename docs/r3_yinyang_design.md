## STEP 1: GD 기획서

### 1. 게임 개요
흑(●)과 백(○)으로 격자를 채우는 이진 퍼즐.
비나이로와 같은 이진 토글 패턴이지만, 핵심 규칙이 다르다:
- **연결성**: 같은 색의 셀은 모두 하나의 연결된 영역을 이루어야 한다
- **2×2 금지**: 같은 색의 2×2 블록이 존재하면 안 된다

### 2. 규칙 정의
#### 필수 규칙
1. N×N 격자의 모든 빈 칸을 흑(●) 또는 백(○)으로 채운다
2. 흑 셀은 모두 상하좌우로 연결되어 하나의 영역을 이루어야 한다
3. 백 셀도 모두 상하좌우로 연결되어 하나의 영역을 이루어야 한다
4. 같은 색의 2×2 블록이 존재하면 안 된다

#### 승리 조건
모든 빈 칸이 채워지고, 위 3개 규칙을 모두 만족

#### 규칙 설명 텍스트
```
1. 모든 칸을 흑(●) 또는 백(○)으로 채우세요
2. 같은 색은 모두 하나의 영역으로 연결되어야 합니다
3. 같은 색의 2×2 블록은 만들 수 없습니다
```

### 7. 힌트 4단계
- Level 1: "행 {R}, 열 {C} 주변을 살펴보세요"
- Level 2: "이 셀을 ●로 채우면 2×2 블록이 생기거나, ○ 영역이 끊어집니다"
- Level 3: "이 셀은 연결성 규칙에 의해 ●(또는 ○)이어야 합니다. {상세 설명}"
- Level 4: 정답 자동 입력
- **힌트에 ●/○ 기호 사용, 숫자 0/1 금지**

### 8. 고유 설정
- 연결 영역 강조 표시: ON/OFF (같은 색 영역 시각화)

## STEP 2: UX 명세서

### 인터랙션: 패턴 A (이진 토글) — 비나이로와 동일
```
입력 모드 선택: ●(흑) / ○(백) / 지우개
셀 탭 → 현재 모드에 따라 배치/제거
```

### 화면 흐름
```
허브 → 음양 홈 → 난이도 선택 → 플레이 → 완료 → 결과 → 홈
```

### 라우팅
- `/yin-yang` → 음양 홈
- `/yin-yang/game` → 플레이

### 보드 레이아웃 — 비나이로와 동일 구조
- 세로: 보드 상단 + 컨트롤 하단
- 가로: 보드 좌측 + 컨트롤 우측

### 셀 렌더링
- ●(흑): 검은 원 (비나이로와 동일 크기 cellSize × 0.32)
- ○(백): 흰 원 (테두리 표시)
- 빈 칸: 비어있음
- 규칙 위반: 2×2 블록 → 빨간 배경, 연결 끊김 → 주황 배경

### 일시정지/결과 화면 — 스도쿠 통일 (공통 키 사용)
### 하드웨어 백키 — 비나이로와 동일
### 접근성/다국어 — 비나이로와 동일 패턴
