# K-Puzzles — 13개의 논리 퍼즐, 하나의 조용한 앱

> 마음을 편안하게, 한 칸씩.

## 1. 앱 소개 (Overview)

**K-Puzzles(K-퍼즐)** 는 전 세계의 검증된 논리 퍼즐 13종을 한국적 감성과 모바일 UX 원칙으로 다시 짠 **통합 퍼즐 플랫폼**입니다. 스도쿠 한 종으로 시작하지 않고, 처음부터 **숫자 · 패턴 · 추론 · 시각** 네 갈래의 퍼즐을 한 앱 안에서 자유롭게 오갈 수 있도록 설계했습니다.

### "K"가 의미하는 것
- **K**orea — 한국에서 만든, 한국적 절제미의 인터페이스
- **K**nowledge — 풀이 기법을 학습으로 환원하는 힌트 시스템
- **K**uzzle (Puzzle) — 13종 퍼즐의 통합 플랫폼

### 다섯 가지 차별점
1. **완전 오프라인** — 앱에 `INTERNET` 권한이 없습니다. 네트워크 호출 0건, 추적 0건.
2. **광고 없음** — 인터스티셜, 보상형, 배너 광고 모두 없습니다. 흐름이 끊기지 않습니다.
3. **8개 언어 + 자동 RTL** — 한국어, 영어, 일본어, 중국어, 힌디어, 스페인어, 아랍어, 프랑스어. 아랍어 선택 시 레이아웃이 자동으로 우→좌로 전환됩니다.
4. **13종 퍼즐 통합** — 각 퍼즐을 별도 앱으로 깔 필요가 없습니다. 통계와 배지는 게임별로 분리 관리됩니다.
5. **4단계 점진 공개 힌트 + 인터랙티브 튜토리얼** — 단순히 정답을 알려주는 힌트가 아니라, "왜 그 답인지"를 단계별로 시각으로 증명합니다.

## 2. 타겟 사용자 페르소나

### 페르소나 A — 입문자 "수민" (20대 후반, 직장인)
- **상황**: 출퇴근 지하철 5~10분, "머리 쓰는 가벼운 게임"을 찾는다.
- **선호 게임**: 스도쿠 입문 난이도, 타쿠주 6×6, 음양 작은 보드.
- **가치 제안**: 인터랙티브 튜토리얼 5단계로 규칙을 50초 안에 익히고, 4×4 미니 스도쿠로 첫 성공을 체험.
- **핵심 매력**: "광고가 안 떠서 한 판 끝낼 때 흐름이 안 깨진다."

### 페르소나 B — 일일 한 판 "지호" (40대, 두 아이 부모)
- **상황**: 하루 한 번, 잠들기 전 10분의 "혼자만의 시간".
- **선호 게임**: 오늘의 퍼즐(매일 자동 출제), 스도쿠 보통/어려움, 카쿠로.
- **가치 제안**: 월간 캘린더에 완료/퍼펙트가 색으로 누적, 연속 출석일(streak) 배지.
- **핵심 매력**: "오늘의 퍼즐 하나만 풀고 자자"는 습관이 부담 없이 유지된다.

### 페르소나 C — 도전자 "현우" (50대, 퍼즐 마니아)
- **상황**: 주말에 깊이 있는 한 판. 기법을 의식적으로 학습한다.
- **선호 게임**: 마스터 난이도 스도쿠(X-Wing 필요), 14×14 타쿠주, 스카이스크래퍼 7×7, 카쿠로.
- **가치 제안**: 클리어 화면의 "사용한 기법" 자동 기록, 기법 사전(Glossary), 도전 모드(실수 3회 제한, 힌트 불가).
- **핵심 매력**: "이번 판에서 Hidden Pair를 썼구나"라는 메타 인지가 다음 판으로 이어진다.

## 3. 핵심 기능

### 4단계 점진 공개 힌트 시스템
스도쿠 기준 — 다른 게임에도 점차 확장 중. 답을 바로 주지 않고 인지 단계에 맞춰 단계별로 공개합니다.
- **L1**: 정답이 풀리는 **영역**을 펄스로 강조 ("어디를 보면 될까?")
- **L2**: 적용 가능한 **기법명**과 핵심 셀에 초록 테두리 ("Hidden Single이 보입니다")
- **L3**: 같은 영역의 **후보 숫자 자동 메모 + 소거 시각화** ("왜 그 답인지" 시각 증명)
- **L4**: **정답 자동 입력 + 글로우 애니메이션**

### 13게임 인터랙티브 튜토리얼
각 게임마다 4~5단계 튜토리얼이 내장되어 있습니다. 단순히 텍스트로 규칙을 나열하는 대신, 미니 보드 일러스트와 손가락 안내 애니메이션으로 보여줍니다. 스도쿠 튜토리얼의 마지막 단계는 4×4 미니 스도쿠 한 칸을 직접 풀어보는 **연습형 인터랙션**으로 끝납니다.

### 8개 언어 + 자동 RTL
한국어(기본), 영어, 일본어, 중국어, 힌디어, 스페인어, 아랍어, 프랑스어 — 약 700개 이상의 i18n 키. 아랍어 선택 시 보드 좌표와 단서 위치가 RTL로 자동 반전됩니다.

### 완전 오프라인, 추적 0
앱 매니페스트에 인터넷 권한이 **선언되어 있지 않습니다**. 백업/복원은 단말 내 JSON 파일로 처리되며, 클라우드 동기화는 의도적으로 제공하지 않습니다. 광고 SDK, 분석 SDK, 크래시 리포트 SDK 모두 미탑재.

### 사용 기법 자동 기록 (스도쿠)
한 판을 클리어하면 결과 화면에 "이번 게임에서 학습한 기법: Hidden Single, Naked Pair"처럼 자동 감지된 기법이 나열됩니다. 탭하면 기법 사전(In-app Glossary)이 열립니다.

### 접근성 & 신경 다양성 친화
- **화면 효과 줄이기**: 모든 애니메이션을 정적 표시로 전환
- **글자 크기 3단계**: 기본 / 크게 / 매우 크게
- **8개 테마**: 라이트, 다크, 카페(브라운 톤), 종이(세피아), 집중(딥 네이비), 고대비 등 — 일부는 게임 완료 시 해금
- **TalkBack 친화 라벨**: 시각 정보를 텍스트로도 함께 제공

## 4. 13게임 상세 소개

### 1. 스도쿠 (Sudoku) — 숫자 논리 퍼즐의 정석
9×9 격자를 1~9로 채우는, 누구나 아는 그 퍼즐. K-Puzzles의 스도쿠는 입문~마스터 6단계 난이도와 노출된 단일 후보, 숨은 페어, X-Wing까지 자동 감지하는 힌트 엔진을 갖췄습니다. 클래식 / 릴렉스 / 오늘의 퍼즐 / 도전 모드 지원. **보드**: 9×9. **평균 클리어**: 보통 8~12분. **타겟**: 입문~고급 전 구간.

### 2. 타쿠주 (Takuzu) — ●과 ○로 채우는 이진 패턴
검은 원과 흰 원만으로 채우는 단순함 속의 깊이. 행/열의 두 기호 수가 같아야 하고, 같은 기호 3개 연속 금지, 동일 행/열 중복 금지의 세 규칙으로 작동합니다. 탭 한 번으로 ● ↔ ○ ↔ 빈칸을 순환시키는 단일 입력 모델. **보드**: 6×6 ~ 14×14. **평균 클리어**: 6×6은 1~2분, 14×14는 20분+. **타겟**: 입문→고급.

### 3. 지뢰찾기 (Minesweeper) — 숫자 단서로 지뢰 찾기
PC 시절의 그 고전을 모바일에 맞게 다듬었습니다. 깃발 모드 토글, 안전 칸 자동 확장, 첫 클릭 안전 보장. 빠르게 한 판 끝내고 싶을 때 가장 좋은 선택. **보드**: 난이도별 가변. **평균 클리어**: 3~10분. **타겟**: 입문~중급.

### 4. 음양 (Yin-Yang) — 흑백 조화의 영역 퍼즐
모든 칸을 검은 돌 또는 흰 돌로 채우되, **같은 색은 모두 하나로 연결**되어야 하며 **같은 색 2×2 블록 금지**라는 두 가지 토폴로지 규칙으로 풀립니다. 숫자 없이 시각 패턴만으로 추론하는 매력. **보드**: 중소형. **평균 클리어**: 5~10분. **타겟**: 패턴 인식형 중급.

### 5. 노노그램 (Nonogram) — 숫자 단서로 그림 완성
행/열 가장자리의 숫자 단서를 따라 칸을 채워 **숨겨진 그림**을 드러내는 퍼즐. 칠하기 모드와 X 표시 모드를 토글로 전환. "단서가 0인 줄은 전부 X"처럼 명확한 결정이 빠르게 누적되는 카타르시스가 있습니다. **보드**: 5×5 ~ 15×15. **평균 클리어**: 5~15분. **타겟**: 시각형 입문~중급.

### 6. 킬러 스도쿠 (Killer Sudoku) — 합과 영역으로 푸는 스도쿠
일반 스도쿠 규칙에 **점선 케이지의 합계 조건**이 더해진 변형. 같은 케이지 내 숫자 중복은 금지되며, "합 15를 4칸으로"처럼 조합론적 사고가 요구됩니다. 스도쿠에 익숙해진 사용자의 다음 도전. **보드**: 9×9. **평균 클리어**: 15~25분. **타겟**: 스도쿠 중급 이상.

### 7. 스타 배틀 (Star Battle) — 별 배치 논리 퍼즐
각 행/열/굵은 선 영역에 **정확히 N개의 별**을 배치하되, 별끼리는 가로/세로/대각선으로 인접할 수 없습니다. 보드 위에 후보를 X로 표시하며 좁혀가는 추론 게임. **보드**: 중형. **평균 클리어**: 7~15분. **타겟**: 영역 추론형 중급.

### 8. 라이트 업 (Light Up) — 전구로 보드 밝히기
검은 칸을 피해 전구를 놓아 **모든 흰 칸을 밝히는** 빛 전파 퍼즐. 전구는 상하좌우로 빛을 보내고 검은 벽에서 멈춥니다. 두 전구가 서로의 빛을 쏘면 안 되고, 숫자 검은 칸은 인접 4방향의 전구 수를 지정합니다. **보드**: 7×7 ~ 14×14. **평균 클리어**: 5~12분. **타겟**: 공간 추론형 중급.

### 9. 부등호 스도쿠 (Futoshiki) — 부등호 단서 라틴 방진
1~N 라틴 방진에 **인접 칸 사이의 < > 부등호 단서**가 추가됩니다. 5×5에서 시작하지만 작은 보드에서도 부등호 체인이 길게 이어지면 의외로 까다롭습니다. 스도쿠와 다른 결의 사고를 요구. **보드**: 5×5 ~ 7×7. **평균 클리어**: 8~15분. **타겟**: 스도쿠 변형 선호 중급.

### 10. 텐트 (Tents) — 나무 옆 텐트 배치
각 나무에는 **정확히 하나의 인접 텐트**가 짝지어져야 하고, 텐트끼리는 가로/세로/대각선으로 인접할 수 없으며, 행/열 가장자리 숫자가 그 줄의 텐트 수를 지정합니다. 친근한 캠핑 모티브와 빡빡한 논리의 조합. **보드**: 8×8 ~ 12×12. **평균 클리어**: 7~15분. **타겟**: 시각 추론형 중급.

### 11. 직소 스도쿠 (Jigsaw Sudoku) — 불규칙 영역 스도쿠
3×3 박스 대신 **불규칙한 모양의 9개 영역**을 사용하는 스도쿠 변형. 영역 경계가 비대칭이라 시야가 분산되며, 같은 단일 후보라도 영역 분할에 따라 전혀 다른 결론이 도출됩니다. **보드**: 9×9. **평균 클리어**: 12~20분. **타겟**: 스도쿠 중상급.

### 12. 스카이스크래퍼 (Skyscrapers) — 시야로 빌딩 높이 맞추기
1~N 라틴 방진에 **가장자리 숫자 = 그 방향에서 보이는 빌딩 수**라는 시야 단서가 더해집니다. 높은 빌딩은 뒤쪽 낮은 빌딩을 가린다는 단순 규칙이 의외로 강력한 제약을 만듭니다. **보드**: 4×4 ~ 7×7. **평균 클리어**: 6~15분. **타겟**: 공간 시각화 중급.

### 13. 카쿠로 (Kakuro) — 합으로 채우는 십자말 *NEW*
가로/세로 단서로 주어진 합계와 일치하도록 1~9를 채우는 **숫자 십자말풀이**. 한 단서 내 숫자 중복 금지. "17을 2칸"이면 8+9로 확정되는 식의 조합 결정이 게임 진행의 묘미. **보드**: 가변. **평균 클리어**: 12~25분. **타겟**: 숫자 조합형 중상급.

## 5. 게임 분류 (사용자 관점)

### 카테고리별
- **숫자형**: 스도쿠, 킬러 스도쿠, 직소 스도쿠, 부등호 스도쿠, 스카이스크래퍼, 카쿠로
- **패턴/이진형**: 타쿠주, 음양
- **추론/배치형**: 스타 배틀, 텐트, 라이트 업, 지뢰찾기
- **시각/그림형**: 노노그램

### 난이도별 추천 진행 순서
1. **시작점**: 스도쿠 (입문) → 타쿠주 6×6 → 지뢰찾기 쉬움
2. **다음 단계**: 노노그램 → 음양 → 텐트 → 라이트 업
3. **응용**: 스타 배틀 → 부등호 스도쿠 → 스카이스크래퍼
4. **고급**: 킬러 스도쿠 → 직소 스도쿠 → 카쿠로 → 스도쿠 마스터

### 한 판 소요 시간별
- **5분 미만**: 타쿠주 6×6, 스카이스크래퍼 4×4, 지뢰찾기 쉬움
- **5~15분**: 스도쿠 보통, 노노그램, 음양, 텐트, 라이트 업, 부등호 스도쿠
- **15분+**: 킬러 스도쿠, 직소 스도쿠, 카쿠로, 스도쿠 마스터, 타쿠주 14×14

## 6. 사회적 가치 / 차별화 메시지

### 데이터 0, 추적 0
앱이 인터넷에 접속하지 않습니다. 이는 마케팅 문구가 아니라 **매니페스트 레벨의 약속**입니다. 어린이, 보안 민감 직군, 데이터 절약 환경 사용자가 안심하고 쓸 수 있습니다.

### 광고 없음 = 흐름 보존
퍼즐은 몰입이 핵심입니다. 한 판이 끝날 때마다 30초짜리 전면 광고가 뜨는 경험을 K-Puzzles는 만들지 않았습니다.

### 두뇌 트레이닝 가치
13게임은 각기 다른 인지 영역을 자극합니다 — 숫자 조합력(스도쿠/카쿠로), 패턴 인식(타쿠주/노노그램), 토폴로지 추론(음양), 공간 시각화(스카이스크래퍼/라이트 업), 제약 만족 탐색(스타 배틀/텐트). 한 앱 안에서 인지 다양성을 누적할 수 있습니다.

### 글로벌 + 한국 정체성
8개 언어로 글로벌 시장에 동시 출시 가능한 폭을 가지면서, "K-" 브랜딩과 한국적 절제미(여백, 톤 다운된 컬러 팔레트, "마음을 편안하게" 카피)로 정체성을 명확히 합니다.

### 신경 다양성 친화
모션 감도, 색 대비, 글자 크기, TalkBack 대응까지 — 접근성을 "추가 기능"이 아니라 **설계 기준**으로 다뤘습니다. ADHD/자폐 스펙트럼 사용자가 "조용한 한 판"으로 쓸 수 있도록.

## 7. 기술 정보

- **프레임워크**: Flutter 3.44 + Dart 3.12
- **상태 관리**: Riverpod, 라우팅 go_router
- **플랫폼**: Android 12+ (minSdk 31, targetSdk 35), iOS 14+ 호환 코드베이스
- **권한**: `INTERNET` 권한 **선언 없음**. 저장소 접근은 백업/복원 시에만 사용자 동의 흐름으로 처리.
- **국제화**: 8개 언어 약 700+ i18n 키, 아랍어 자동 RTL.
- **빌드**: 적응형 아이콘, AAB 배포 (Play Store Alpha 운영 중).
- **아키텍처**: PuzzleEngine 인터페이스 + GameRegistry 플러그인 구조 — 새 게임 추가는 폴더 + 등록만으로 가능.

## 8. 로드맵 / 향후 계획

- **힌트 시스템 확장**: 스도쿠 외 게임에도 4단계 점진 공개 힌트와 기법 사전을 확대 적용.
- **기법 기반 배지**: 사용 기법 자동 기록(H4 인프라)을 기반으로 "X-Wing 마스터", "Naked Triple 10회" 등 학습형 배지 추가.
- **학습 모드**: 특정 기법만 등장하는 연습 퍼즐 큐레이션.
- **통계 심화**: 게임별 시간 추이 그래프, 난이도별 그레이드 분포.
- **추가 게임 후보**: 슬리더링크, 누리카베 등 — 플러그인 구조 덕에 게임 단위로 안전하게 추가 가능.

> 한 판 끝낼 때 광고가 뜨지 않는, 인터넷이 없는 비행기에서도 똑같이 동작하는, 데이터가 새지 않는 퍼즐 앱. **K-Puzzles는 그 약속에서 시작합니다.**
