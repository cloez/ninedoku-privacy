## 1. 게임 개요

### 1.1 설명
숫자 힌트를 이용해 지뢰의 위치를 논리적으로 추론하는 퍼즐. 
전통적인 지뢰찾기와 달리, **모든 퍼즐이 찍기(추측) 없이 순수 논리만으로 풀이 가능**하도록 설계한다.

### 1.2 인터랙션 패턴
**패턴 A 변형 (이진 토글)**: 탭으로 셀 열기, 길게 누르기로 깃발(⚑) 배치/제거

### 1.3 핵심 차별점
- **논리적 풀이 보장**: 찍기 없는 지뢰찾기 (No-guess Minesweeper)
- **첫 클릭 안전**: 첫 번째로 여는 셀은 반드시 안전 (숫자 0, 주변 8칸도 안전)
- **시드 기반 생성**: 같은 시드 → 같은 퍼즐 (오늘의 퍼즐, 재현 가능)

## 2. 규칙 정의

### 2.1 필수 규칙
1. N×N 격자에 지뢰가 숨겨져 있다
2. 셀을 열면 주변 8방향 지뢰 개수가 숫자로 표시된다
3. 주변에 지뢰가 없으면(0) 인접 빈 셀이 연쇄적으로 열린다
4. 모든 지뢰가 아닌 셀을 열면 승리한다
5. 깃발(⚑)로 지뢰로 추정되는 위치를 표시할 수 있다 (선택적)

### 2.2 선택 규칙 (설정으로 제어)
- **실수 표시**: 지뢰를 열었을 때 즉시 게임 오버 vs 실수 카운트만 증가
  - 기본값: 실수 카운트 (게임 오버 없음, 논리 퍼즐 모드)
- **남은 지뢰 수 표시**: 전체 지뢰 수 - 깃발 수 = 남은 수

### 2.3 승리 조건
- 지뢰가 아닌 모든 셀을 연 경우 (깃발은 승리 조건에 포함되지 않음)

### 2.4 규칙 설명 텍스트 (30초 이해)
```
1. 셀을 탭하면 열립니다. 숫자는 주변 8칸의 지뢰 개수입니다
2. 길게 누르면 깃발(⚑)을 세워 지뢰 위치를 표시합니다
3. 숫자 힌트를 활용해 모든 안전한 셀을 열면 승리합니다
```

## 3. 난이도 체계

### 3.1 격자 크기 + 지뢰 비율

| 난이도 | 격자 | 지뢰 수 | 지뢰 비율 | 빈 셀 수 |
|--------|------|---------|----------|----------|
| 입문 | 8×8 | 8 | 12.5% | 56 |
| 쉬움 | 9×9 | 12 | 14.8% | 69 |
| 보통 | 10×10 | 18 | 18.0% | 82 |
| 어려움 | 12×12 | 30 | 20.8% | 114 |
| 마스터 | 16×16 | 50 | 19.5% | 206 |

### 3.2 풀이 기법 수준
- **입문**: 단순 셀 열기 (숫자 1 주변에 닫힌 셀 1개 → 확정 지뢰)
- **쉬움**: 기본 소거 (숫자와 깃발 수 비교)
- **보통**: 패턴 인식 (1-2-1 패턴, 코너 패턴)
- **어려움**: 영역 교차 논리 (두 숫자 셀의 영향 범위 교차)
- **마스터**: 고급 추론 (가능/불가능 배치 분석)

## 4. 등급 기준 (S/A/B/C)

### 4.1 등급 정의
```
S (Perfect): 실수 0 + 힌트 0 + 기준 시간 이내
A (Excellent): 실수 ≤ 1 + 힌트 ≤ 1 + 기준 시간 × 1.5 이내
B (Great): 실수 ≤ 3 + 힌트 ≤ 3 + 기준 시간 × 2 이내
C (Good): 완료하면 부여
```

### 4.2 난이도별 기준 시간

| 난이도 | 기준 시간 | S 기준 | A 기준 | B 기준 |
|--------|----------|--------|--------|--------|
| 입문 | 60초 | ≤ 60초 | ≤ 90초 | ≤ 120초 |
| 쉬움 | 120초 | ≤ 120초 | ≤ 180초 | ≤ 240초 |
| 보통 | 300초 | ≤ 300초 | ≤ 450초 | ≤ 600초 |
| 어려움 | 600초 | ≤ 600초 | ≤ 900초 | ≤ 1200초 |
| 마스터 | 1200초 | ≤ 1200초 | ≤ 1800초 | ≤ 2400초 |

### 5.1 클래식 모드
- 난이도 선택 → 퍼즐 생성 → 풀이
- 시간/실수 카운트 표시
- 등급 산정

### 5.2 릴렉스 모드
- 시간 표시 없음, 실수 카운트 없음
- 등급 산정 없음 (완료만 기록)

### 5.3 오늘의 퍼즐
- 날짜 기반 시드 → 전 사용자 동일 퍼즐
- 보통 난이도 (10×10, 18지뢰)
- 하루 1회만 도전 가능

## 7. 힌트 4단계

### Level 1: 어디를 봐야 하는지
```
"행 {R}, 열 {C} 주변을 살펴보세요"
→ 해당 셀 위치 강조 표시
```

### Level 2: 무엇을 생각해야 하는지
```
"이 숫자({N}) 주변에 닫힌 셀이 {M}개 있습니다"
→ 관련 셀들 강조 표시
```

### Level 3: 왜 그런지 (풀이 기법 설명)
```
"이 셀의 숫자가 {N}이고 주변에 이미 ⚑가 {F}개 있으므로, 
남은 닫힌 셀 중 {N-F}개가 지뢰입니다. 
안전한 셀을 열 수 있습니다."

또는:
"이 셀의 숫자 {N}과 주변 닫힌 셀 수 {M}이 같으므로,
모든 닫힌 셀이 지뢰(⚑)입니다."
```

### Level 4: 정답 공개
```
→ 안전한 셀 자동 오픈 또는 지뢰 위치에 ⚑ 자동 배치
```

**주의**: 힌트 텍스트에 "지뢰", "⚑"(깃발) 등 게임 고유 용어/기호를 사용. 
숫자 0, 1 등 추상적 표현 금지.
