# 한국어(_ko) UX 라이팅 리뷰

대상: `lib/shared/l10n/app_strings.dart` — _ko 영역 약 700 키

## 1. 종합 점수: 8.2 / 10

전반적으로 자연스러운 한국어이며 정중체 톤이 안정적이다. 다만 (a) 자동 추가 키 일부의 조사·맞춤법 오류, (b) 게임명 "~란?" 받침 처리 누락, (c) 영문/한글 혼용("Coming Soon"), (d) 너무 짧게 축약된 게임 내 버튼 라벨이 약점이다.

## 2. 우수한 카테고리

- **홈/온보딩/튜토리얼**: 부드러운 정중체, 카피 라이팅 톤이 잘 잡혀 있음 (`home.subtitle`, `onboarding.page1.desc`).
- **배지 명칭**: "첫 걸음", "독립 해결사", "꾸준한 시작" 등 감성적이고 짧음.
- **설정/통계**: 명사형 라벨이 일관적이며 모바일 폭에 적합.
- **스도쿠 규칙 설명**: 정확하고 군더더기 없음.

## 5. 톤 일관성

- **존댓말 정책**: 사용자 대상 메시지는 "~합니다 / ~하세요" 정중체, 버튼은 명사형(예: "새 게임", "포기")로 분리되어 일관적이다. **유지 권고.**
- **불일치 지점**:
  - 게임별 일시정지 메뉴 — 스도쿠는 `포기하기`, 비나이로는 `포기` (BR액션 길이 차). 모든 게임에서 `포기하기`로 통일 권장.
  - "이어하기" vs "재개" 혼용 (`game.resume`=`계속하기`, `binairo.pause.resume`=`재개`). **`계속하기`로 통일** 권고.
  - 영문 잔재: `mode.comingSoon`, `hub.new`(`NEW`) — `NEW`는 배지 컨벤션으로 허용 가능하나 한글 `신규` 검토 가능.
- **명사형 라벨 길이**: 게임 내 툴바 4~5글자 이내 잘 지켜짐. 단 `자동메모`/`셀우선`은 띄어쓰기만 보완하면 더 가독.
