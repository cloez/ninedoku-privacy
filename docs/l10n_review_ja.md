# 日本語ローカライズレビュー (_ja)

対象: `lib/shared/l10n/app_strings.dart` — `_ja` マップ全キー (約 380 件)

## 1. 総合評価: **7.8 / 10**

全体的に自然で、モバイル UI に適した簡潔さを保っている。常体（敬体混在）も統一感あり。ただし「諦める／ギブアップ」「退出／戻る／離れる」「ハブへ／ゲームハブへ」など**ゲーム間の用語ゆれ**が顕著で、後半に自動追加された不等号スドク以降のゲーム群で品質がやや低下する。

## 2. 優れているカテゴリ

- **基礎 UI / ホーム / 設定**: `home.subtitle`「穏やかに、一マスずつ」など情緒的かつ短く秀逸。
- **チュートリアル (`tutorial.page*`)**: 自然な説明文、句読点の改行配置も適切。
- **オンボーディング**: 「心を落ち着ける数独ゲームです」など親しみやすさが高い。
- **数独本体のバッジ名**: 「最初の一歩」「独力解決」「完璧主義者」など意訳が巧み。

## 5. トーン一貫性

- **敬体/常体**: ボタンは常体（「戻る」「続ける」）、説明文は敬体（「〜します」「〜できます」）で概ね統一。良好。
- **語尾の「。」**: ルール文 (`*.rules.r1` 等) は句点あり、UI ラベルは句点なしで一貫。
- **漢字比率**: 約 35–40% で読みやすい範囲。「諦める」「退出」のみ硬すぎる印象。
- **半角/全角混在**: 疑問符 `?`／`？` がゲーム間で混在（要全角統一）。

## 6. コード直接修正は行っていません

本レビューは観察と推奨のみ。実際の文字列変更は DEV エージェントの担当範囲です。

### 한국어 요약

전반적으로 자연스러운 일본어 품질(7.8/10)이며 홈·튜토리얼·온보딩 카피는 우수합니다. 단, 후반에 자동 추가된 게임군(不等号스도쿠/킬러/카쿠로 등)에서 「諦める」(→ギブアップ), 「退出」(→終了), 「ゲームハブへ」(→ハブへ), 「〜スドク」(→〜ナンプレ 또는 数独 표기) 같은 용어 불일치가 다수 발견됩니다. P0 5건 우선 통일 시 브랜드 일관성이 크게 개선됩니다. 의문부호 반각/전각 혼재도 전수 정리 권고합니다.
