4. 음양 — 흑백 돌의 연결과 배치로 푸는 영역 퍼즐.

11. 직소 스도쿠 — 불규칙한 9개 영역의 스도쿠.
12. 스카이스크래퍼 — 시야 단서로 빌딩 높이를 추론.

## 4. 다국어(현지화) 메모

- 앱이 이미 8개 언어를 지원하므로, Play Console 등록정보도 **최소 영어**는 별도 현지화 등록 권장 (글로벌 노출/전환에 직접 영향).
- 우선순위: 한국어(기본) → 영어 → 일본어 → 중국어 순으로 등록정보 번역 추가 검토.
- 각 언어 등록정보는 Play Console > 스토어 등록정보 > "언어 추가"에서 개별 입력.

## 5. 변경 시 주의 (이름 변경 영향)

- 앱 이름(나인도쿠 → K-퍼즐) 변경은 등록정보 텍스트 변경이며 **앱 업데이트(새 AAB) 없이도 가능**. 단, 앱 내부 표시 이름과 일치시키려면 새 빌드 반영 권장.
- 패키지명(applicationId `com.cloez.sudoku`)은 **변경 불가** — 그대로 유지.
- 이름 변경 후 스토어 반영까지 검토 시간이 걸릴 수 있음.
