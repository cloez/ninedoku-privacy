## C. motionScale 동작

| 항목 | 결과 | 비고 |

| `MediaQuery.disableAnimationsOf` 감지 | PASS | 0.0 반환 |
| `reduce_effects` SharedPreferences 키 감지 | PASS | 0.3 반환 |
| 기본 1.0 | PASS | |
| `scaledDuration` 헬퍼 | PASS | scale=0 시 `Duration.zero` |
| 공통 위젯 모두 motionScale 적용 | PASS | AnimatedCell / CountUpText / AnimatedTrophy / BoardProgressBar 4종 모두 사용 |

**경미한 이슈**: `motionScale`은 `prefs` 인자를 직접 받음. 위젯들이 `widget.prefs`를 전달하지 않으면 reduce_effects가 항상 false 처리 — 게임 화면에서 SharedPreferences를 위젯에 명시 전달해야 동작. 현재 통합되지 않아 게임 화면에서는 시스템 접근성만 동작하고 앱 내 설정 토글은 효과 없음.

## I. APK 크기 영향

- `audioplayers ^6.0.0` 패키지: 예상 +0.3~0.6MB (Android ExoPlayer 기반)
- `assets/sounds/*.ogg` 7개: **현재 0 byte → 영향 0**
- 실제 자산 교체 후: 7 × ~50KB ≈ +350KB 예상
- 총 예상 증분: **+0.7~1.0MB** (자산 교체 후 기준)
