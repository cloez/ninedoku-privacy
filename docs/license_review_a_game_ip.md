## 2. 주요 게임 IP 근거

### Binairo 🟡 (유일한 주의 게임)
- **상표 주체**: Conceptis Ltd. (이스라엘) — 자사 신디케이션 퍼즐 브랜드로 "Binairo" 명칭 사용
- **등록 여부**: 미국 USPTO 정식 등록 여부 **확인 필요**. 다만 Conceptis는 "Pic-a-Pix", "Link-a-Pix" 등 다수 등록 보유 → 동일 패턴으로 "Binairo"도 상표 주장 가능성 높음
- **일반명 대안**: **Takuzu**, **Binary Puzzle**, **0h h1** (모두 상표 자유)
- **룰**: 0/1 균등 배치 + 3연속 금지 + 행/열 유일성 → 룰 자체는 idea, 저작권 보호 안 됨

### Sudoku 🟢
- Nikoli가 1984년 "数独(Sūdoku)" 명명 후 일본 상표 등록, 그러나 전 세계 보호 실패 → 일반명사화. Conceptis·앱스토어 무수한 사용 사례. 안전.

### Nonogram 🟢 / Picross 회피
- **Picross**: Nintendo/Jupiter Corp. 등록 상표 → 사용 금지
- **Griddler**: Conceptis 상표 → 사용 금지
- **Nonogram**: 西尾徹也 학자가 1980년대 명명한 일반 학술명. 안전.

### Light Up 🟢 / Akari 회피
- **Akari (あかり)**: Nikoli 등록 상표 → 사용 금지
- **Light Up**: 영문권 일반명. Simon Tatham's Puzzles 등 다수 사용. 안전.

### Killer Sudoku 🟢
- 1997년 독일(Sumdoku) → 2005년 The Times(영국) 영문 도입. **상표 등록 확인되지 않음**. Conceptis·Nikoli·앱스토어 광범위 사용. 안전.

### Kakuro 🟢
- Nikoli 1986년 명명("加算クロス" 축약). 미국 Dell Magazine은 1966년부터 "Cross Sums"로 발간. 현재 양 명칭 모두 상표 자유. 안전.

## 3. 권고 사항

### P0 (출시 전 필수 조치)
1. **Binairo → Takuzu 또는 Binary Puzzle 명칭 변경 검토**
   - 위험: Conceptis가 침해 주장 시 앱 스토어 takedown 가능
   - 비용: ARB 키 다국어 4개 + 아이콘 라벨만 변경. 엔진 무관
   - 권장 명칭: **"Takuzu"** (일본어 起源 일반명, 위키피디아 표준 표기)
   - 대안: 부제 표기 — "Binary Puzzle (Takuzu)"

### P1 (안전망)
2. 모든 게임 설명에 **"이 게임은 전통 퍼즐을 기반으로 합니다"** 문구 추가 → 룰 기원의 공공성 명시
3. 앱 스토어 설명/스크린샷에 **™, ®, Nikoli, Conceptis, Nintendo, Picross, Akari, Griddler, Pic-a-Pix** 단어 사용 금지
4. **Killer Sudoku**: 영국·일본 시장 한정 출시 시 The Times 측 클레임 사례 모니터링 (현재까지 보고 없음)
5. 아이콘·UI에서 타사 등록 마크(특히 Nintendo 픽토그램, Nikoli 로고 폰트) 미사용 재확인

## 4. 종합 판단

**출시 가능 여부**: ✅ **출시 가능** — 단, **P0 Binairo 명칭 변경 권고**
