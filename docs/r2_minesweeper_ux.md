## 2. 인터랙션 상세

### 패턴 A 변형 (이진 토글 기반)

**기본 동작:**
- **탭**: 셀 열기 (안전한 셀 → 숫자/빈칸 표시, 지뢰 → 실수 처리)
- **길게 누르기 (500ms)**: 깃발(⚑) 토글 (닫힌 셀에만)

**입력 모드 컨트롤 바:**
```
[👆 열기] [⚑ 깃발] [↩ 되돌리기]
```
- 열기 모드: 탭 → 셀 열기
- 깃발 모드: 탭 → 깃발 배치/제거
- 되돌리기: 마지막 동작 취소

**연쇄 오픈:**
- 숫자 0인 셀 열기 → 주변 8칸 자동 오픈 → 재귀적 확장
- 파동 애니메이션 (100ms 딜레이 per depth level)

**셀 상태:**
```
닫힌 셀 (미탐색) → 배경색 어두운 톤
열린 셀 (숫자) → 배경색 밝은 톤, 숫자 표시
열린 셀 (빈칸) → 배경색 밝은 톤
깃발 셀 (⚑) → 닫힌 배경 + ⚑ 아이콘
지뢰 (실수) → 빨간 배경 + 💣 아이콘 (실수 표시 모드일 때)
```

**숫자 색상:**
```
1: 파란색 (#2196F3)
2: 초록색 (#4CAF50)
3: 빨간색 (#F44336)
4: 남색 (#3F51B5)
5: 갈색 (#795548)
6: 청록색 (#009688)
7: 검정 (#212121)
8: 회색 (#9E9E9E)
```

## 3. 보드 레이아웃

### 세로 모드
```
┌─────────────────────┐
│ AppBar (지뢰찾기)     │
│ 타이머 | 실수 | ⚑남은수│
├─────────────────────┤
│                     │
│    지뢰찾기 보드      │
│    (정사각형 격자)     │
│                     │
├─────────────────────┤
│ 힌트 메시지 영역       │ ← AnimatedSize 적용
├─────────────────────┤
│ [👆열기] [⚑깃발] [↩] │
│ [💡힌트]              │
└─────────────────────┘
```

### 가로 모드
```
┌────────────────────────────────────┐
│ 상단 바 (← | 지뢰찾기 | ⏸)         │
├──────────────────┬─────────────────┤
│                  │ 타이머/실수/남은수 │
│   지뢰찾기 보드    │ 힌트 메시지       │
│                  │ [👆] [⚑] [↩]    │
│                  │ [💡힌트]          │
└──────────────────┴─────────────────┘
```

### 셀 크기 계산
- 세로 모드: `min(화면너비, 화면높이 × 0.55) / gridSize`
- 가로 모드: `화면높이 × 0.85 / gridSize`
- 최소 셀 크기: 28dp (16×16 마스터용)
- 최대 셀 크기: 48dp (8×8 입문용)

## 4. 하드웨어 백키 동작

| 화면 | 백키 동작 | 비고 |
|------|----------|------|
| 지뢰찾기 홈 | 무시 (PopScope canPop: false) | 허브 이동은 apps_rounded 아이콘 |
| 플레이 중 | 나가기 확인 다이얼로그 | AppBar ← 와 동일 |
| 일시정지 | resume() | 게임 재개 |
| 결과 화면 | 게임 홈으로 이동 | |

---

## 5. 일시정지 화면 (스도쿠 통일)

```
버튼 3개:
1. 재개 (ElevatedButton) — pause.resume 키
2. 홈으로 (자동 저장) (OutlinedButton) — pause.home 키
3. 포기 (TextButton.icon, 빨간색) — pause.giveUp 키
```
- "허브로" 표기 금지
- 보드 가림 처리 (오버레이)
- 하드웨어 백키 → resume()

---

## 6. 결과 화면 (스도쿠 통일)

```
1. 등급 배지 (S/A/B/C + 색상)
2. 통계 (시간, 난이도, 실수, 힌트, 열린 셀 수)
3. 새로 획득한 배지 칩 섹션
4. 버튼: 새 게임(ElevatedButton) → 홈(OutlinedButton)
```
- ConsumerStatefulWidget
- initState에서 새 배지 팝업 자동 표시

## 7. 게임 홈 화면

### AppBar
- 타이틀: "지뢰찾기" (game.minesweeper.name 키)
- Leading: Icons.apps_rounded (허브 이동)
- Actions: 통계/배지 버튼

### 컨텐츠
- 이어하기 카드 (진행 중 게임 있을 때)
- 규칙 설명 카드 (_RulesHint, 빈 상태일 때)
- 모드 선택 (클래식 / 릴렉스 / 오늘의 퍼즐)
- 난이도 선택 (클래식/릴렉스 선택 후)

### 통계/배지 연동
- 통계 버튼: `extra: 'minesweeper'` → 해당 게임 탭 자동 선택
- 배지 버튼: `extra: 'minesweeper'` → 해당 게임 탭 자동 선택

## 8. 접근성

- 셀 Semantics: "행 {r} 열 {c}, {상태}" (닫힘/열림/숫자/깃발/지뢰)
- 셀 최소 터치 영역: 44×44dp (입문~보통), 28dp까지 허용 (마스터)
- 텍스트 크기 배율 대응 (1.0x, 1.3x, 1.6x)
- 숫자 색상: 고대비 보장 (배경과 최소 4.5:1)

---

## 9. 다국어 문자열 키 전체 목록

### 4개국어 작성 필수
(기획서 §9의 키 목록 + 아래 추가)

- `minesweeper.mode.classic` / `minesweeper.mode.relax` / `minesweeper.mode.daily`
- `minesweeper.input.open`: 열기
- `minesweeper.input.flag`: 깃발
- `minesweeper.setting.game_over_on_mine`: 지뢰 열면 게임 오버
- `minesweeper.setting.show_remaining`: 남은 지뢰 수 표시
- `minesweeper.setting.cascade_animation`: 연쇄 오픈 애니메이션
